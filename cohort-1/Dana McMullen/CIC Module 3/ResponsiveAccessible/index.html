Final accessibility and design assignment

Instructor: Zoe K.

Due: April 5th, 2019

Design a mobile responsive front page of a magazine or a magazine article. It should be accessible and include heuristics and design principles that we have gone over this week.

It should include the following:

  • Grouped content with (Title, Author, Photo, Main text) -- decide on an html hierarchy that is accessible and change the design in css if it makes more sense on all devices)
  • At least one form input and a button to sign up for a newsletter
  • Semantic HTML
  • Good use of color and typography
  • Styled with Sass

    • You should display your elements in an accessible hierarchy and use css to display the visual design the way you want. Keep in mind device agnostic design / reusable patterns that was discussed in the video by Ethan Marcotte.

    If you create your site on Glitch, send me the link. If you create it with a text editor, zip the files and attach them.
